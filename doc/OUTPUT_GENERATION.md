# Output Generation

Phase 5 transforms analytical findings into human-readable and visual reports.

## Components

### Visualization Agent
- **Description**: The visual architect.
- **Responsibilities**:
    - Selects the optimal chart types (Bar, Line, Radar, etc.) based on data characteristics.
    - Shapes raw numerical data into JSON structures ready for frontend charting libraries.

### Report / Narrative Agent
- **Description**: The storyteller.
- **Responsibilities**:
    - Generates executive-level summaries.
    - Highlights "Key Takeaways" and "Actionable Insights".
    - Adjusts tone and language to match the target audience (Technical vs. Executive).
