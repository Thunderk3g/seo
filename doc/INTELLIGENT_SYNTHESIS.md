# Intelligent Synthesis

Phase 4 is where the core data processing and reasoning occur.

## Components

### Semantic Umbrella Agent
- **Description**: Merges disparate data streams.
- **Responsibilities**:
    - Resolves semantic conflicts (e.g., when two APIs report different values).
    - Identifies "Missing Data" gaps and requests secondary fetches if necessary.
    - Contextualizes observations within the user's specific industry.

### Analysis / Reasoning Agent
- **Description**: The "Brain" of the system.
- **Responsibilities**:
    - Trend Detection: Identifies patterns over time.
    - Root Cause Analysis: Explains *why* metrics are changing.
    - Correlation: Finds hidden links between different data points.
